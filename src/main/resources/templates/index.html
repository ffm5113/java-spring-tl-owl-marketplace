On the Web Listing (OWL) Marketplace

By PSU IST 440W Group 7 - Summer 2022


On the Web Listing (OWL) Marketplace was created as a capstone project for Pennsylvania
State University's Information Sciences and Technology Integration and Problem Solving
undergraduate course. This property listing application project was expanded upon by
Forrest Moulin, who integrated an Azure MySQL Database for flexible server, an Azure Web
App Service, and Bing Maps API. Furthermore, OWL Marketplace is a fictitious, non-profit
organization, and the application code is open-source, provided under the MIT License.
